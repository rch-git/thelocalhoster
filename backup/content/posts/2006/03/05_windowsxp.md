title: 'How Long Will Windows XP Last?'
---
Speaking personally, developments over the last few months have been quite grim.

PC hardware is changing like never before. And with the release of Windows Vista, a complete change in computer lifestyle seems inevitable.

But that question that has been nagging me, how long can Windows XP last?

Vista is bringing with a vast number of changes, and I am not too sure I can keep up with this change. Rumor has it that the operating system is going to cost anywhere between $400-$500, convert that into Indian money, and it's an insane price range of 17,710 - 22,137.50 Rupees. I could buy a new computer with that price!

However, the price might well justify the quality of the product. But if the price is anywhere close to those figures, I will have to stay with Windows XP for a long time.

I have used Windows 95, 98, ME, 2000, XP, and I liked XP the best. I am so very comfortable with it right now, that even the promise of a better operating system will not convince me easily to migrate.

Microsoft have withdrawn support for Windows 98, I don't know when this has happened, but it did. However, I believe there is still support for Windows 2000 and XP.

I hope it stays that way for a long time to come, becuase Windows XP is Microsoft's best operating system to date, and I wouldn't want it to be outdated anytime soon.
